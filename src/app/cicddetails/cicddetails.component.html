🚀 Growing as a Cloud DevOps Enthusiast: Building Automated & Scalable Solutions

As a passionate Cloud DevOps student, I’m diving deep into automation and continuous integration to streamline software delivery. I focus on creating smart CI/CD pipelines using tools like GitHub Actions and Jenkins ⚙️, enabling faster, more reliable deployments and helping teams deliver value efficiently.

🧠 My Expert Areas where I gained skill 🛠️

I explore cloud infrastructure on platforms such as AWS ☁️, Azure ☁️, and Google Cloud ☁️, learning how to design secure, scalable environments by managing resources like VPCs, IAM policies 🔐, and serverless functions ⚡. I also work with OpenStack 🖥️ for private cloud setups.

Containerization with Docker 🐳 and orchestration with Kubernetes ☸️ are key skills I’m developing to build flexible, manageable microservices architectures that scale with demand.

🔍 Monitoring, Security & Development Skills

I’m passionate about monitoring and security, implementing tools like Prometheus 📊, Grafana 📈, and the ELK stack 🐘, while embracing DevSecOps principles 🔒 to integrate security throughout the development lifecycle.

On the development side, I’m sharpening my skills with full stack projects using Angular ⚛️ for frontend and Spring Boot 🌱 for backend, aiming to build seamless, performant applications.

I optimize pipelines to run parallel jobs when possible, reducing overall build and deployment time ⏱️⚡.
